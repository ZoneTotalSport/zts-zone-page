📖 COMMENT UTILISER LE PLANIFICATEUR

1. CONFIGURE TES PÉRIODES

Clique sur une case du calendrier → le modal s'ouvre → clique sur ⚙️ Configuration → choisis le nombre de périodes (3 à 7), les heures, les récréations et le dîner → "Appliquer à tout le calendrier".

2. PLANIFIE CHAQUE JOUR

Clique sur un jour → remplis les groupes (ex: 204B), les activités (ex: Basketball) et les détails pour chaque période. Ajoute tes gardes de transition entre les périodes.

3. AJOUTE CONGÉS & EVENTS

Utilise les boutons en bas du calendrier pour ajouter des congés (sur plusieurs jours!), des journées pédagogiques, des sorties scolaires. Les congés fériés du Québec sont déjà là!

4. EXPORTE & IMPRIME

Exporte ta planification en PDF, Excel ou CSV — par jour ou par mois complet. Imprime pour afficher au gymnase ou partager avec la direction!

💡 ASTUCES

  • Les points colorés (vert/bleu/violet) apparaissent quand tu écris "cycle 1", "c2", etc. dans les groupes
  • Le bouton "📋 Planif" apparaît dans la case quand tu as sauvegardé une planification
  • Tout est sauvegardé automatiquement dans ton navigateur (localStorage)
  • 4 langues disponibles : français, anglais, espagnol et chinois mandarin
  • Ajoute tes extras du jour : parascolaire, comité, et autre dans le modal
